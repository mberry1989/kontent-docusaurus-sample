Adjusting site properties like **title**, **tagline**, **urls**, **favicons**, and **plugins **can be done via the ***docusaurus.config.js*** file found in the root of the site.

For example, take a look at [this site's configuration](https://github.com/kentico-michaelb/kontent-docusaurus-sample/blob/master/my-website/docusaurus.config.js "GitHub Kontent + Docusaurus configuration").
