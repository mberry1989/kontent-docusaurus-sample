[Clone](https://docs.github.com/en/free-pro-team@latest/github/creating-cloning-and-archiving-repositories/cloning-a-repository "Cloning a repository") the [Kontent + Docusaurus repository](https://github.com/kentico-michaelb/kontent-docusaurus-sample "Kontent + Docusaurus repository") with your favorite GIT client

    OR

[Fork](https://docs.github.com/en/free-pro-team@latest/github/getting-started-with-github/fork-a-repo "Fork a repo") the repository if you would like to contribute to submit Pull Requests at a later date

:::note Fork vs. Clone

The difference between "forking" and "Cloning" GitHub repositories is explained in [this post](https://github.community/t/the-difference-between-forking-and-cloning-a-repository/10189 "The difference between forking and cloning a repository") on GitHub's community page.

:::
